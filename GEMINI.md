# Gemini Agent

## Work Agreement

- User provides a task to complete
- LLM uses [AGENTS.md](AGENTS.md) as global guidance.
- LLM analyzes the task and attempts to complete it autonomously
- LLM is doing it's best effort to complete the task without additional user intervention
- LLM may ask for clarification in case of ambiguous or incomplete tasks, however it should be a rare occurrence
